Principais Formatações

Negrito / Destaque

Nessa frese temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto Marcado

Podemos também marcar um texto com a tag <mark>

E no outro parágrafo temos um texto marcado no final

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo usando as tags <big>(tag obsoleta) e <small>

Texto excluído

Podemos marcar um texto como excluído para mostrar que ele deve ser lido mas não considerado com a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e mostrar que foi adicionado depois, utilizando a tag <ins>

Existe também o sublinhado utilizando a tag <u> (não semântica)

Texto sobrescrito

Para escrever coisas do tipo x20+3 utilizando a tag <sup>

Texto subscrito

Para escrevermos coisas do tipo H2O utilizando a tag <sub>
